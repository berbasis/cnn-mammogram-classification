# cnn-mammogram-classification
This program classifies and localizes breast cancer in mammogram images into non-malignant (normal or benign) and malignant using Convolutional Neural Networks (CNN). 
- Dataset used in this project is the INbreast public dataset which can be downloaded from here: http://medicalresearch.inescporto.pt/breastresearch/index.php/Get_INbreast_Database
- Trained InceptionV3 model can be downloaded from here: https://drive.google.com/file/d/1ehrTFrUMZIV_1LhrUOrBEwAv79sl29Pu/view?usp=sharing
- Journal (Bahasa Indonesia) can be downloaded from here: https://drive.google.com/file/d/1udgLto2XjVcg1YF2YX8qQFNnZRhng4RF/view?usp=sharing
